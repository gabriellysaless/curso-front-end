# Aula prática CSS-in-JS

Esse projeto foi feito para uma aula prática do curso de Full-Stack da EBAC.
- Foi estruturado um componente de interface chamado CardProduto, onde refatoramos seu estilo usando **React e Styled Components**
- Implementação de renderização dinâmica de produtos com .map()
- Estilização dinâmica baseada em props (alteração de cor e texto do botão)
- Organização modular do projeto (componentes e estilos separados)

O que é o Lorem Ipsum?

O Lorem Ipsum é um texto fictício usado na área de design e tipografia como preenchimento provisório em layouts. Sua função é simular o espaço que será ocupado por um conteúdo real, ajudando a visualizar a estrutura e o aspecto estético de um projeto antes de ter o texto definitivo. Ele é derivado de um escrito em latim de Cícero, datado do século I a.C., mas não possui sentido completo — justamente para não distrair o leitor e manter o foco no design.
